Hardware.js

Hardware.js is a framework to enable IoT Webapp products and solutions. It is based on modern HTML5 browser and hardware plugins.

IoT Webapp

  • Develop IoT webapp using HTML5, CSS3 and Javascript
  • Access to hardware by Javascript
  • Collaboration oriented IoT solutions

Easy to implement IoT Webapp

You can simply add the required hardware package e.g. <script src="zkgpio.js"></script>. You can then access the hardware by the provided JavaScript functions. e.g GPIO pins.

Implemented Hardware plugins for Raspberry Pi

  • zkgpio.js - GPIO
  • zkpifacecad.js - PiFace Control and Display
  • MORE coming soon!
